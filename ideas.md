# AdCraft AI - Design Brainstorm

## ที่เลือก: Dark Premium Tech (Probability: 0.08)

### Design Movement
**Neo-Brutalist Tech Minimalism** - ผสมผสานความเด่นของ Dark Mode ร่วมกับ Accent Color ที่เด่นชัด (Red/Crimson) เพื่อสร้างความรู้สึก Premium, Modern, และ Conversion-Focused

### Core Principles
1. **Contrast-Driven**: ใช้พื้นหลังดำ (#0b0b0b) เป็นพื้นฐาน ทำให้ Accent Color (แดง #ff2d2d) ดูเด่นและสำคัญมากขึ้น
2. **Functional Minimalism**: ลบส่วนที่ไม่จำเป็น เน้นเฉพาะ CTA, Input, Output ที่ชัดเจน
3. **Tech Credibility**: ใช้ Grid Pattern บาง ๆ เป็น Background Texture เพื่อสร้างความรู้สึก "Advanced AI Platform"
4. **Conversion-First**: ทุก Element ต้องมีวัตถุประสงค์ชัดเจน ไม่มี Decoration ที่ไร้ประโยชน์

### Color Philosophy
- **Primary**: #0b0b0b (Pure Black) - ความเชื่อถือ, ความเป็นมืออาชีพ
- **Accent**: #ff2d2d (Vibrant Red) - Urgency, Action, Conversion
- **Secondary**: #1a1a1a, #2a2a2a (Dark Grays) - Depth, Hierarchy
- **Text**: #ffffff (White), #e0e0e0 (Light Gray) - Readability
- **Glow**: Subtle Red Shadow/Glow บน CTA Buttons - Premium Feel

### Layout Paradigm
- **Asymmetric Hero**: ซ้าย = Copy + CTA, ขวา = Visual/Screenshot
- **Sidebar Navigation**: สำหรับ Dashboard (Dark Sidebar + Light Content)
- **Card-Based Grid**: Templates & Designs ใช้ Dark Cards กับ Red Border/Hover
- **Two-Column Generate**: ซ้าย = Input Panel, ขวา = Live Preview

### Signature Elements
1. **Red Glow Button**: CTA Buttons มี Subtle Glow Effect, Hover = Brighter Glow
2. **Grid Background**: Thin Grid Pattern บน Hero & Sections เพื่อสร้าง Tech Feel
3. **Red Accent Line**: Divider/Separator ใช้ Red Line แทน Border ธรรมดา

### Interaction Philosophy
- **Instant Feedback**: ทุก Click ต้องมี Visual Response (Glow, Scale, Color Change)
- **Smooth Transitions**: 0.3s ease-in-out สำหรับ Hover, Focus, Active States
- **Progressive Disclosure**: Input Panel ซ่อนรายละเอียดเพิ่มเติมจนกว่า User จะต้องการ

### Animation
- **Button Hover**: Scale 1.02 + Glow Intensify
- **Card Hover**: Border Color Red + Subtle Lift (Shadow)
- **Page Transition**: Fade In (300ms) เมื่อ Load Page
- **Loading State**: Spinner ที่มี Red Color + Rotation Animation

### Typography System
- **Display Font**: Poppins Bold (700) - Headline, Hero Title
- **Body Font**: Inter Regular (400) - Body Text, Description
- **Accent Font**: Poppins SemiBold (600) - Button Text, Labels
- **Hierarchy**: H1 (48px) → H2 (32px) → H3 (24px) → Body (16px)

## ไม่เลือก: Option 2 - Gradient Minimalism (Probability: 0.07)

### Design Movement
**Glassmorphism with Gradient Accents** - ใช้ Frosted Glass Effect + Red-to-Black Gradient เพื่อสร้างความรู้สึก Modern & Futuristic

### Core Principles
1. **Transparency & Depth**: ใช้ Backdrop Blur + Semi-transparent Cards
2. **Gradient Flow**: Red → Black Gradient ไหลผ่าน Hero & Key Sections
3. **Soft Edges**: ทุก Element มี Rounded Corners (16px+)
4. **Layered Depth**: Multiple Layers ของ Cards & Shadows

## ไม่เลือก: Option 3 - Bold Typography Focus (Probability: 0.06)

### Design Movement
**Brutalist Typography** - ใช้ Large Bold Text + Minimal Decoration เพื่อสร้างความรู้สึก Strong & Direct

### Core Principles
1. **Typography First**: ข้อความเป็น Hero ของ Design
2. **Extreme Contrast**: Black + White + Red เท่านั้น
3. **Geometric Shapes**: ใช้ Simple Shapes (Circles, Lines) แทน Complex Graphics
4. **Whitespace**: ใช้ Whitespace เพื่อ Breathing Room

---

## Implementation Notes
- ใช้ Poppins + Inter Fonts จาก Google Fonts
- ทุก CTA Button ต้องมี Red Color + Glow Effect
- Grid Background Pattern สร้างจาก CSS (ไม่ใช่ Image)
- Dark Mode เป็น Default Theme
- Responsive Design: Mobile First Approach
